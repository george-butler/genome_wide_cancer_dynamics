# MCMC modelling pipeline 

## Running the MCMC pipeline

The MCMC pipeline can be run via: 

Rscript model_fit.R


## Changing the MCMC pipeline for different genes
The pipeline can be applied to different genes by changing line 359. Note, the path in line 359 should point towards the screen output from aBSREL that is automatically saved by absrel_fit.sh (file ending _absrel.txt).
